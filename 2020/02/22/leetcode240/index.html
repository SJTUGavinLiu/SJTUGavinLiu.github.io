搜索二维矩阵

编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target。该矩阵具有以下特性：

每行的元素从左到右升序排列。
每列的元素从上到下升序排列。

分治-二分分割

思路与实现

将原矩阵十字划分，根据中心点的大小决定需要处理哪三个子矩阵。

  1. matrix[midx][midy] == target
  2. matrix[midx][midy] < target 除了右下角的矩阵均要遍历
  3. matrix[midx][midy] > target 除了左上角的矩阵均要遍历

在搜索对应矩阵前先判定目标是否在矩阵内！

复杂度分析

分治-线性分割

思路与实现

我们沿着中间列找到行号row满足matrix[row-1][mid] < target < matrix[row][mid]。此时在行row，列mid的分割下分别有四个子矩阵，显然左上角，右下角矩阵都不可能包含目标。原问题就被拆分成了两个子矩阵的问题。

复杂度分析

简单分析两种极端情况

贪心

思路与实现

和leetcode209思路类似，我们可以从左下角出发，遵循如下规则查找

  1. 当前位置大于目标，向上
  2. 当前位置小于目标，向右

行列的递增性保证这样的查找不会错过目标

复杂度分析

时间复杂度$O(m+n)$
